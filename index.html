Sobre Este Site

Este projeto foi criado para centralizar os trabalhos da disciplina de Design de Interação. A cada nova atividade, uma nova página será adicionada ao menu para manter todo o conteúdo organizado em um único lugar.

A proposta é mostrar a evolução das técnicas de HTML, CSS e JavaScript ao longo do semestre, com foco em estrutura semântica, usabilidade, design responsivo e interatividade.
